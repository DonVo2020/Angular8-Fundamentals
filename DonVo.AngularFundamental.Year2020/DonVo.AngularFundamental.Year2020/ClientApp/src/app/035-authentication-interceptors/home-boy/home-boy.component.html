This component is public, i.e. no authentication required

This project builds upon work by Jason Watmore. See below. Added is the option to observe the status (login/logout) in the upper right corner of the screen as an observable.

If you try to navigate to the Random Users page, you are redirected to the login page if you're not logged in.

This project consists of two sub-projects. They both have a package.json and you need to perform npm install for both of them!

  • /client - the Angular-application using authentication and interceptors
  • /server - a simple NodeJS server using JWT-authentication

Server

  • Go to the /server folder and perform npm install.
  • Open a command prompt in the folder and type npm start.
  • This will start a simple NodeJS server, listening for incoming requests at http://localhost:4000

Client

It is this project. You are already running it.
